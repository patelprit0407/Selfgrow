# SelfGrow — Personal Growth Tracker

A lightweight, fully offline personal growth OS built with vanilla HTML, CSS & JavaScript.

## Features
- ✅ Daily habit tracker with 7-day view
- 📊 Weekly progress stats & streak counter
- 📝 Weekly review journal (auto-saved)
- 🧠 Personality & trait analysis
- 💾 Zero backend — all data stored in your device's localStorage
- 📱 Works on mobile, tablet & desktop

## Tech Stack
- Pure HTML + CSS + JavaScript
- No frameworks, no database, no server
- localStorage for data persistence
